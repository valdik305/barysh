# GREEN-API Demo Page

Демонстрационная страница для работы с GREEN-API WhatsApp API.

## 🚀 Функциональность!

- `getSettings` - получение настроек инстанса
- `getStateInstance` - получение состояния инстанса
- `sendMessage` - отправка текстового сообщения
- `sendFileByUrl` - отправка файла по ссылке

## 🔧 Использование

1. Зарегистрируйтесь на [GREEN-API](https://console.green-api.com)
2. Создайте инстанс и подключите WhatsApp
3. Скопируйте `idInstance` и `ApiTokenInstance`
4. Введите их в соответствующие поля на странице
5. Используйте кнопки для тестирования методов API

## 🌐 Демо

Страница доступна по адресу: `https://[username].github.io/[repo-name]/`
